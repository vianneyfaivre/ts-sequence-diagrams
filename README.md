## Build the grammar

Build the grammar using Jison:

```
npm install jison -g
npm run build-grammar
```

To export the generated code as a module, update the end of SequenceDiagram.js:
```
Parser.prototype.parse = parser.parse;
return new Parser;
})();
export default SequenceDiagram;
```

## Run

```
npm run start
```
